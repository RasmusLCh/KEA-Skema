How do i assign a user to a group?

Assigning users to groups is done in the infrastructure.

Add one or more students to a course

  • In the infrastructure add the users to a group.
  • Open Course admin
  • Click on Manage Course
  • Click edit
  • Select the student groups that needs access to the course
  • Click edit

What is Subject prioritizer

Subject prioritizer is used, when a teacher uploads a schedule, with different subjects. for example

Class today, Homework, Further reading material

Here til teacher can assign different prioritizes for what is show first forexample

Class today (priority = 200)
Homework (priority = 100)
Further reading material (priority = 50)

With the Subject Prioritzer, you can specific that any subject that matches the ones you have defined, gets a specific priority. Forexample if you set Further reading material to priority = 500, the list would be

Further reading material (priority = 500)
Class today (priority = 200)
Homework (priority = 100)
The higher the priority the sooner the subject will be shown, the max priority is 1000.

What is the webcrawler?

The webcrawler scrabes data from keaplan.kea.dk, this page is linked with an internal system that KEA uses for assigning rooms for courses.
When the webpage is scrabed, rooms and teacher for specific lectures are updated.
If lectures are found on keaplan.kea.dk they are added to the course that matches the name.
If no course is found, a new course is created for the lectures. Any ground matching course name is added as students to the course.
