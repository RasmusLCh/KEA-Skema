Admin mode

Welcome administrator! Below you can find a short description of what is available to you in admin mode.
If you cannot find your answer below, please send us an email support@mor.it

Users

Users in the system, the users are authenticated by logging into Office 365 SMTP server.

Groups

Groups in the system, groups can contain other groups or users. The max depth of a group is 50.

Import

Imports users or groups from Outlook address book, please use these scripts to generate a CSV/JSON file to import:
Get users Get groups When importing users or groups, already created users or groups will be updated - Users or groups in the system will not be deleted, if they are not in the CSV/JSON file.

Statistics

Show how much the system is being used.

Services

Her you can set, who can access what microservices based on the groups you have created.

All service options, File Resources, Action, Page Injections and Top Menu Links are only available to users that has access to the service.

Service Options

Her you can define specific settings the users can set, that your services are using.

File Resources

Files that the services are showing to the users.

Actions

Alert services, when a specific actions happens in the infrastructure, for example when an user is created.

Page Injections

Inject css og js into specific pages, to change their design og the structure of the page with javascript.

Top Menu Links

Top menu links that are available on the main page.
