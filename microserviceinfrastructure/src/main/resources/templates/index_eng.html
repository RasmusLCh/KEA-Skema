Welcome!

Hi and welcome to KEA learning, a place for student and teacher to organize their learning. Currently you can see your schedule, homework and your attendance in the system.

To get started, please log into the system.

Feels like the system is missing a key feateure?

If you feel like the system is missing a keyfeature, come help us, the system is 100% open source and easy to extend. That means that you can develope extensions (called microservices) to the system and integrate them easily and with out risk to the rest of the system! Checkout the github repository for more information and how to get started!

Things that aren't working / suggestions

The system is still in beta, so if you find anything that is not working or you got suggestions, let us know!
